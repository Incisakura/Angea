# Angea

---
> Naming from hydrangea(アジサイ)

A lite tool to make systemd work in any container(Windows Subsystem for Linux 2, Docker, Podman, etc.)

**WSL1 is not supported.**

## Usage

See `angea help`

## Requirement

No! But you should install `systemd` as least.
